# zotero_summarizer

A new Python project.

## Setup

1.  Create the virtual environment (if not already present):
    ```bash
    uv venv
    ```

2.  Activate the virtual environment:
    ```bash
    source .venv/bin/activate
    ```

3.  Install dependencies:
    ```bash
    uv pip install -r requirements.txt
    ```
